# MediCore – Hospital Management System

MediCore is a **modern full-stack hospital management platform** built with **.NET 10 and Blazor**.  
The goal of the project is to simulate a **production-grade healthcare SaaS system** used by hospitals and clinics to manage patients, medical records, pharmacy inventory, and administrative operations.

The platform demonstrates **enterprise software architecture**, a **modern admin dashboard UI**, and advanced features such as **analytics dashboards, PDF medical reports, and real-time notifications**.

---

# Project Goals

MediCore is designed as a **portfolio-quality enterprise application** to demonstrate:

- Modern **.NET backend architecture**
- **Blazor dashboard UI development**
- **API-driven modular systems**
- Healthcare **Electronic Medical Records (EMR) modeling**
- **Pharmacy inventory management**
- **Analytics dashboards**
- **Real-time notifications**
- Clean separation of domain, infrastructure, and UI layers
